## 📦 Установка

<details>
<summary><b>composer.json</b> (нажмите, чтобы развернуть)</summary>

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/integrat/amocrm-library"
        }
    ],
    "require": {
        "integrat/amocrm-library": "^1.0"
    }
}
```

</details>

Или выполните пошагово:

**1. Добавьте репозиторий в `composer.json`:**
```json
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/integrat/amocrm-library"
    }
]
```
